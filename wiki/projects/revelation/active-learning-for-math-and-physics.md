# Active Learning For Math And Physics

Revelation needs an explicit practice layer because advanced math and physics
are unusually vulnerable to spectator-mode learning. Beautiful exposition can
feel productive while leaving no usable skill behind.

The minimum standard is that each study session produces work: exercises,
derivations, toy examples, summaries, diagrams, prerequisite notes, or
questions sharpened enough to answer later.

## Texts Are Maps, Exercises Are Territory

Use expository texts to orient, but use exercises to test whether the
orientation is real.

For any section, choose at least one active output:

- solve a book exercise;
- rederive a central equation;
- invent a nearby example;
- explain a theorem or postulate in different words;
- compute a toy case;
- write a two-page synthesis note;
- identify the exact prerequisite blocking progress.

Problem-heavy texts are useful when a topic needs muscle rather than more
motivation. They should be paired with conceptual texts rather than replacing
them.

## Just-In-Time Prerequisites

The current source notes repeatedly warn against prerequisite hell. The better
pattern is depth-first and local:

1. Continue the main text until a missing concept blocks understanding.
2. Write the missing concept into a prerequisite scratchpad.
3. Learn the minimal usable version needed for the current passage.
4. Return to the spine.
5. Deepen the prerequisite only if it recurs.

This avoids the false obligation to finish entire prerequisite books before
touching the motivating question.

## Fatigue And Session Design

Mathematical fatigue is expected, not a failure signal. Dense notation,
abstract concepts, unfamiliar prerequisites, and cumulative cognitive load all
make reading stop abruptly.

Useful responses:

- shorten sessions before comprehension collapses;
- take microbreaks instead of abandoning the day;
- switch from reading to one small exercise or summary;
- use alternate explanations only after identifying the exact blockage;
- stop with a restart note: where to resume, what is confusing, and what the
  next action is.

The goal is not maximal session length. The goal is reliable restart and
compounding understanding.

## Weekly Rhythm

A practical Revelation week can have three modes:

- spine sessions: slow reading, derivations, exercises;
- satellite sessions: just-in-time geometry, math, or background;
- playground sessions: toy systems, conceptual probes, and synthesis notes.

The playground is not optional. It is where the beauty and mystery motivations
stay alive instead of being buried under textbook coverage.

## AI Assistance

AI tools should be used as tutors, not answer engines.

Good uses:

- graded hints;
- alternate explanations after a first attempt;
- checking derivations;
- generating adjacent problems;
- summarizing prerequisite concepts;
- producing small computational experiments.

Risky uses:

- reading full solutions before struggling;
- substituting summaries for text;
- letting the agent decide the learning path without source-grounded review;
- over-expanding into many parallel topics.
