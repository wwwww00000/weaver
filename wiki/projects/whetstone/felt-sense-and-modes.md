# Felt Sense And Modes

Felt sense is the compressed, partly nonverbal signal that something is right,
wrong, promising, tense, open, overfit, alive, or dead. In Whetstone, the
useful version is not mysticism. It is a handle for fast pattern recognition,
somatic markers, predictive error signals, and domain-specific intuition.

The goal is to make those signals trainable enough to calibrate, without
pretending they are already precise.

## Expertise Handles

Felt-sense labels should work like small UI icons:

- tight or open;
- pull or push;
- spin or settle;
- sharp or fog;
- click or smear;
- alive or dead.

These labels are useful when they point to action. "Smear" might mean the
representation is muddy. "Tight" might mean overcontrol. "Click" might mark a
compressed model worth testing. The label should lead to a check, contrast,
or next experiment.

## Domain Feel

Domain feel can be treated as a fast internal simulator with calibrated error
signals. A domain-specific felt sense improves when it is trained against
examples, counterexamples, perturbations, and feedback.

Useful ingredients include:

- prototypes: canonical examples and near misses;
- invariants: what should stay true under transformation;
- perturbation intuition: what changes when one knob moves;
- error signatures: how wrongness feels in this domain;
- control policy: which move to try when a signal appears.

This makes felt sense compatible with rigorous work. The signal suggests where
to look; evidence still decides.

## Modes Of Consciousness

The mode vocabulary source distinguishes analytic, embodied, affective,
spontaneous, felt-sense, associative, exploratory, narrative, observer, flow,
open-monitoring, social, and sensory modes.

Chronicle can use this vocabulary for meditation, journaling, and state
awareness. Whetstone can use it for control: notice whether the current mode
fits the task, then loosen, focus, embody, narrate, or switch representations.

The more operational register is simpler: autopilot, intentional, diffuse,
anxious, imaginative, flow, reflective, and depleted. That register belongs in
[Attention And State Regulation](attention-and-state-regulation.md), where each
label points to a small switch operator.

The coarse categories are more reliable than fine-grained labels. Terms like
heart-softening, spine-brightening, cranial lightness, or widening semantic
radius are useful as first-person handles, not as settled scientific entities.

## Calibration Practices

The practice should be contrastive and evidence-seeking:

- take a two-second somatic ping before a task;
- predict whether the current frame will work;
- solve in two passes, one intuitive and one explicit;
- compare similar cases with different outcomes;
- keep a tiny calibration journal of signal, action, and result;
- use perturbation drills to learn how the felt sense changes.

The review question is simple: did the label predict anything useful, or did it
only create a satisfying story?

## Open Questions

- Which felt-sense labels are stable across domains, and which are domain
  specific?
- How should calibration be measured without over-formalizing introspection?
- Which modes support creative writing versus mathematical or coding work?
- When should a felt signal be trusted, and when should it trigger explicit
  verification?
