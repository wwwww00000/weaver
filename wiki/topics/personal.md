# Personal

This topic collects personal-practice material that does not fit cleanly into a
named project such as Chronicle, Whetstone, Conjuration, Genesis, or Accretion.

Use this area sparingly. If a note is really about journaling, cognition,
creative practice, building, or finance, it should live under the relevant
project. This topic is for cross-cutting life-practice pages that need a home
but are not themselves a named project.
