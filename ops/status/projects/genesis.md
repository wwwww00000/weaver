# Genesis Status

## Current Focus

Building things, exercising agency, agentic tooling, AI-adjacent work, and AI
research. Current center is practical build loops, small AI research loops, and
the tooling layer that turns ideas into artifacts.

## Recent Changes

- Genesis was synthesized into a project hub with pages for agentic tooling, AI
  research, reasoning/tool use, credit assignment, and agency.
- The initial Weaver ingest and synthesis pass is complete.
- Project status cards are being introduced as the live state layer.
- Weaver has been promoted from a Genesis child page to a top-level project.
- Structured context was synthesized as a Genesis/Weaver research thread.

## Decisions

- Treat Weaver as a top-level sibling project. Genesis can still own broader
  agentic tooling and AI-assisted interface work.
- Keep AI research hobby-scale and artifact-oriented.
- Use markdown, git diffs, and task templates as the repeatability layer before
  introducing heavier orchestration.

## Next Actions

- Evaluate whether the next Genesis build should be a notebook controller,
  command palette, agent harness, AI research experiment, or product prototype.
- Decide whether structured context starts as a Weaver sidecar or separate
  Genesis prototype.
- Keep design questions explicit before adding automation.

## Open Questions

- Should subagents or SDK jobs be introduced now, or only after repeated manual
  review passes expose a stable task shape?
- What frontmatter and semantic markers should project pages and status cards
  standardize?
- What should be the first standalone Genesis artifact after the initial
  Weaver bulk pass?

## Blocked Or Waiting

- Waiting on design decisions about automation boundaries, project dashboard
  assembly, and which agentic tooling build should come first.

## Promote To Wiki

- Stable workflow decisions.
- Status-card schema changes that should become project documentation.
- Agent synthesis/review patterns that survive repeated use.
- New build artifacts or research experiments.
