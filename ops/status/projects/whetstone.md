# Whetstone Status

## Current Focus

Cognition improvement through metacognitive control, problem-solving operators,
attention/state regulation, cadence, mental pages, felt-sense calibration, and
skill-acquisition loops.

## Recent Changes

- Whetstone was synthesized into a project hub with child pages for think tags,
  problem solving, attention/state regulation, cadence, and felt sense.
- Cognitive unassigned queues were mined into Whetstone and Chronicle pages.

## Decisions

- Whetstone owns transferable cognitive machinery.
- Chronicle owns lived journaling, mindfulness, and life-review practice.
- Whetstone drills should be measured against transfer or real task usefulness,
  not just subjective intensity.

## Next Actions

- Pick one Whetstone drill to run for a short period.
- Define the review signal before starting the drill.
- Use current active work, such as math, P12n, writing, or software building,
  as the calibration domain.

## Open Questions

- Which Whetstone drills have measurable transfer?
- How much metacognitive tagging is helpful before it disrupts flow?
- Which domain should supply the first calibration tasks?
- What lightweight log format captures useful cognitive experiments?

## Blocked Or Waiting

- Needs a concrete current task domain before the next drill can be selected.

## Promote To Wiki

- Drill protocols that survive actual use.
- Transfer observations from real work.
- Refined think-tag vocabulary that improves decisions rather than adding
  noise.
