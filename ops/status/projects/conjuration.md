# Conjuration Status

## Current Focus

Imagination building, creativity, drawing, and the bridge between inner
visualization and external marks. Current center is constrained practice:
visualization drills, constructive drawing, sourcebook worldwork, and small
creative prompts.

## Recent Changes

- Conjuration was synthesized into a project hub with pages for visualization,
  constructive drawing, drawing studio stack, sourcebook worldwork, and fiction
  premise material.
- Drawing, art, and creativity residual queues were routed into Conjuration.

## Decisions

- Do not treat Conjuration as generic "be more creative" aspiration.
- Anchor sessions in concrete constraints and feedback: draw, compare, rotate,
  construct, or capture.
- Route software tools for drawing or visualization to Genesis if they become
  actual builds.

## Next Actions

  scene study, portrait construction, sourcebook fieldwork, or perspective

- Define a repeatable weeknight Conjuration session.
